# NyxGo项目简介
 - NyxGo项目本身是为**NyxBot**定制的引导脚本,需一定基础,与**NyxBot**环境配置关联,该项目会在/usr/lib文件夹里面创建一个虚拟环境防止污染系统,脚本提供QQ机器人框架的安装以及**NyxBot**的启动配置

# 须知
 - 暂时仅支持**Debian**系系统，例如Debian本体/Ubuntu/Kali/Mint等等以**apt**作为包管理器的**标准Linux系统**

# 用法
 - 启动,使用命令`sudo bash ./start.sh`以启动,**期间注意网络通畅**
