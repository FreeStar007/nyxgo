# NyxGo项目简介
 - **NyxGo**项目本身是为**NyxBot**定制的引导脚本,需一定基础,与**NyxBot**环境配置关联,该项目会在**/usr/lib**文件夹里面创建一个名为**nyxbot_venv**的虚拟环境防止污染系统，脚本提供**QQ机器人框架**的安装以及**NyxBot**的启动配置

# 须知
 - 暂时仅支持***Debian系***以及***RedHat系***的***标准Linux系统***
 - ***Debian系***例子：***Debian本体/Ubuntu/Kali/Mint***等等以***apt***作为包管理器的***标准Linux系统***
 - ***RedHat系***例子：***RedHat本体/CentOS/Fedora/Rocky***等等以***dnf/yum***作为包管理器的***标准Linux系统***

# 用法
 - 启动,使用命令`sudo bash ./start.sh`以启动,***期间注意网络通畅***
